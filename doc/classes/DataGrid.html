Parameters:

  • [options] Object optional

    Configuration object

    • [autoRemoveHandling] Boolean optional

      raises an event before a row is removed

    • [fieldsData] Array optional

      fields data will extend url and set tableHead automatically

      • [{}] Object optional
        fields object
      • [{}.id] String optional
        field name
      • [{}.translation] String optional
        translation key which will be translated automatically
      • [{}.disabled] String optional
        either 'true' or 'false'
    • [editable] Boolean optional

      will not set class is-selectable to prevent hover effect for complete rows

    • [className] String optional

      additional classname for the wrapping div

    • [data] Object optional

      if no url is provided (some functionality like search & sort will not work)

    • [defaultMeasureUnit=px] String optional

      the unit that should be taken

    • [elementType=table] String optional

      type of datagrid (currently only table is available)

    • [excludeFields=[id] Array optional

      array of field to exclude

    • [pagination=false] Boolean optional

      display a pagination

    • [paginationOptions] Object optional

      Configuration Object for the pagination

      • [pageSize] Number optional
        Number of items per page
      • [showPages] Boolean optional
        show pages as visual numbers
    • [pageSize] Number optional

      lines per page

    • [showPages] Number optional

      amount of pages that will be shown

    • [removeRow] Boolean optional

      displays in the last column an icon to remove a row

    • [selectItem] Object optional

      Configuration object of select item (column)

      • [type] String optional
        Type of select [checkbox, radio]
      • [width] String optional
        Width of select column
    • [sortable] Boolean optional

      Defines if list is sortable

    • [columns] Array optional

      configuration array of columns

      • [content] String optional
        column title
      • [width] String optional
        width of column
      • [class] String optional
        css class of th
      • [type] String optional
        type of the column. Used to manipulate its content (e.g. 'date')
      • [attribute] String optional
        mapping information to data (if not set it will just iterate of attributes)
    • [appendTBody] Boolean optional

      add TBODY to table

    • [searchInstanceName=null] String optional

      if set, a listener will be set for the corresponding search event

    • [columnOptionsInstanceName=null] String optional

      if set, a listener will be set for listening for column changes

    • [url] String optional

      url to fetch data from

    • [paginationTemplate] String optional

      template for pagination

    • [validation] Boolean optional

      enables validation for datagrid

    • [addRowTop] Boolean optional

      adds row to the top of the table when add row is triggered

    • [startTabIndex] Boolean optional

      start index for tabindex

    • [progressRow] Boolean optional

      has to be enabled when datagrid is editable to show progress of save action

    • [columnMinWidth] String optional

      sets the minimal width of table columns

    • [contentContainer] String | Object optional

      the container which holds the datagrid; this options resizes the contentContainer for responsiveness

    • [showElementsSteps] Array optional

      Array which contains the steps for the Show-Elements-dropdown as integers

    • [fullWidth] String optional

      If true datagrid style will be full-width mode
